SAT Mistake Inputter

Technologies Used

HTML
CSS
Javascript
Python

Description

This is a tool I made for an SAT bootcamp that I run with some friends to help students record and analyze the mistakes that they made on practice tests. After reviewing questions missed if they are comfortable with the problem they color it green and the question goes to a different column. When inputting new missed questions the column being used is organized and the specific question is appended to the end. Based on the number of questions missed in a section the percent likelihood of getting a certain score is presented. The sidebar was created using HTML and CSS with the logic being executed with Javascript. Python was used to data scrape the curves for the past 13 SATs for each section from the College Board in order to give the students an accurate score based on their performance.
